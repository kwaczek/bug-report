Widget Test Page

This page tests the bug report widget with Bootstrap loaded to verify CSS isolation.

The floating red bug button should appear in the bottom-right corner. Click it to open the report form.

Expected behavior:

  • Bug button appears without Bootstrap style conflicts
  • Form opens with current URL pre-filled
  • Auto-screenshot attempts (may show unavailable on local file)
  • Ctrl+V paste of an image works in the form
  • Submitting shows "Network error" (no backend running) — this is correct
  • Host page is unaffected after any widget error
